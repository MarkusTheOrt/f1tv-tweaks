# Better F1TV

This extension is developed to counter small annoyances with the F1TV streaming page.

# Features
The extension provides the following short-cuts to manage the web player:

| Key | Action |
|-----|--------|
|`f`|Toggle fullscreen|
|`space`| Toggle play/pause |
|`m`|Toggle mute|
